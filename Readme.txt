Author:
	Robi(Main Language: Taiwan Mandarin/Traditional Chinese)
Description:
	To classify by Photo Date.

發布資料:
	__pycache__:
		裡面放打包過的原始檔(需要安裝Python)。
	Dist:
		裡面放.exe執行檔(不需要安裝Python)。
	Source code:
		放置Python原始碼供檢視、指教。
	
使用方法：
	(1)沒有安裝過Python核心以及相關模組的使用者。
		請使用Dist資料夾當中的PhotoDate.exe，並將*.JPG照片檔跟該程式放在同一個資料夾。
		僅需要用滑鼠點兩下PhotoDate.exe即可執行，並將照片依照"拍攝日期"建立資料夾並且歸檔。
		最後亦可以得到一份"Info.csv"文件，可以透過Excel打開查看照片列表。

	(2)有安裝過Python核心與相關模組的使用者。
		請使用__pycache__資料夾當中的PhotoDate.cpython-36.pyw，並將*.JPG照片檔跟該程式放在同一個資料夾。
		僅需要用滑鼠點兩下PhotoDate.cpython-36.pyw即可執行，並將照片依照"拍攝日期"建立資料夾並且歸檔。
		最後亦可以得到一份"Info.csv"文件，可以透過Excel打開查看照片列表。
		
	(3)各方前輩與老師
		請檢視Source code資料夾中的PhotoDate.py檔案，並給予建議與指教，謝謝。
